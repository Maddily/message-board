# Message Board

A simple community message board web app built with Node.js, Express, and EJS. Users can post messages, view all messages, and read individual messages in a corkboard-inspired interface.

## Features

- Post new messages with your name
- View all messages on a visually engaging corkboard
- Click on a message to view its details
- Responsive and accessible design
- Random pastel colors for message notes and pins
- Semantic HTML and basic SEO best practices

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:

   ```sh
   git clone git@github.com:Maddily/message-board.git
   cd message-board
   ```

2. Install dependencies:

   ```sh
   npm install
   ```

3. Start the server:
   ```sh
   npm start
   ```
   The app will run on [http://localhost:3000](http://localhost:3000).
